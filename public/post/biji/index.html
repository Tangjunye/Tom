Tom的Linux学习笔记

以下为Linux课堂笔记

一、Linux是什么？

Linux，全称GNU/Linux，是一种免费使用和自由传播的类UNIX操作系统，其内核由林纳斯·本纳第克特·托瓦兹于1991年10月5日首次发布，它主要受到Minix和Unix思想的启发，是一个基于POSIX的多用户、多任务、支持多线程和多CPU的操作系统。它能运行主要的Unix工具软件、应用程序和网络协议。它支持32位和64位硬件。Linux继承了Unix以网络为核心的设计思想，是一个性能稳定的多用户网络操作系统。

二、Linux主要命令

  1. ls-l：表示list，表示以详细列表的形式进行展示
  2. ls-a：表示显示所有的文件/文件夹（包含了隐藏文件/文件夹）
  3. ls-h：以可读性较高的形式显示文件大小
  4. mkdir 路径 创建目录
  5. touch 文件名 创建一个或多个文件
  6. cp 被复制的文档路径 文档被复制到的路径 复制文件到指定目录
  7. mv 需要移动的文档路径 需要保存的位置路径 移动文件到一个目录下
  8. rm-f 需要移除的文档路径 强制删除文件
  9. zip 打包压缩后文件名 要打包压缩的文件 打包压缩文件
  10. unzip 要解压文件名 解压文件
  11. shutdown -h now 立刻关机
  12. shutdown -r now 立刻重启
  13. cd / 切换到根目录
  14. cd /usr 切换到根目录下的usr目录
  15. cd ../ 切换到上一级目录 或者 cd ..
  16. cd ~ 切换到home目录
  17. cd - 切换到上次访问的目录
  18. 基本上vi可以分为三种状态，分别是命令模式、插入模式和底行模式ESC 退出编辑模式到命令行模式；控制光标移动：↑，↓，j，删除当前行：dd，进入编辑模式：i ，进入底行模式：:；退出编辑： :q，强制退出： :q!，保存并退出： :wq。 查看文件权限rwx：r代表可读，w代表可写，x代表该文件是一个可执行文件
  19. which命令的作用是在PATH变量指定的路径中，搜索某个系统命令的位置，并且返回第一个搜索结果。
  20. find命令在目录结构中搜索文件
  21. sudo是为所有想使用root权限的普通用户设计的。可以让普通用户具有临时使用root权限的权利。
  22. pwd 查看当前目录路径
  23. kill 进程号 结束进程
  24. ifconfig：查看网卡信息
  25. chmod 权限 修改文件权限
  26. groupadd group_name 创建一个新用户组
  27. groupdel group_name 删除一个用户组
  28. useradd user1 创建一个新用户
  29. userdel -r user1 删除一个用户
  30. apt-get install 包名 安装/更新一个 deb 包
  31. cat 直接输出文件的全部内容
  32. head 输出文件开头的内容、-n 指定行数
  33. tail 输出文件末尾的内容，-n输出尾n行
  34. more 按页显示文本文件内容，只允许向前浏览
  35. less 与more类似，允许向前向后浏览
  36. tee 将文件的输出复制到另一个文件
  37. od 将文件内容以指定的禁止显示，-t指定进制
  38. 标准输入文件即0，使用<或«
  39. 标准输出文件即代码1，使用>或»
  40. 标准错误输出文件即代码2，使用2>或2»
  41. xargs 一般是和管道一起使用 -n 选项多行输出
  42. -d 选项可以自定义一个定界符
  43. xargs 的一个选项 -I，使用 -I 指定一个替换字符串 {}，这个字符串在 xargs 扩展时会被替换掉，当 -I 与 xargs 结合使用，每一个参数命令都会被执行一次
  44. xargs 结合 find 使用 find . -type f -name “*.log” -print0 | xargs -0 rm -f
  45. clear 清楚屏幕信息
  46. echo xx 显示文本
  47. date 显示日期和时间
  48. cal 显示当前日期
  49. last 显示最近登录系统的用户信息-6列
  50. history （10） 显示历史指令
  51. sudo adduser lilei sudo 给普通用户赋予root权限
  52. alias l=’ls’ 定义命令别名
  53. unalias l 删除别名
  54. alias 列出别名
  55. ps -ef | grep sshd 查找指定ssh服务进程
  56. ps -ef | grep sshd | grep -v grep 查找指定服务进程，排除gerp身
  57. ps -ef | grep sshd -c 查找指定进程个数
  58. wc：利用 wc 指令我们可以计算文件的 Byte 数、字数、或是列数,若不指定文件名称、或是所给予的文件名为“-”,则 wc 指令会从标准输入设备读取数据。 -c 或–bytes 或–chars 只显示 Bytes 数。 -l 或–lines 只显示列数。 -w 或–words 只显示字数。
  59. linux可通过crontab设置定时任务，常用参数-e 设置任务，-l 查看任务计划，设置任务>的时间格式为：分钟 小时 日期 月份 星期 命令
  60. pvcreate /DEV 创建物理卷
  61. pvs 查看物理卷
  62. pvremove /DEV 删除物理卷
  63. vgcreate VGNAME /DEV 创建卷组
  64. vgextend /DEV 扩展卷组
  65. vgreduce VGNAME /DEV 收缩卷组
  66. vgremove VGNAME 删除卷组
  67. lvcreate -L 10G -n LVNAME VGNAME 创建指定逻辑卷大小和逻辑卷名称的名称
  68. lvs 查看逻辑卷
  69. lvextend -L +10G /DEV 增加10G逻辑卷大小
  70. lvremove /DEV/VGNAME/LVNAME 删除逻辑卷
  71. 脚本shell命令 $ 变量符 ‘’ 单引号字符，全部普通字符 “” 双引号字符，可以包括特殊字符如反引号 $? 返回上一个变量返回值 $# 传送给shell程序位置参数的数量 $0 当前执行的进程名 $! 后台运行的最后一个进程号 $$ 当前进程的进程号 $n 第n个参数 $@ 全部位置参数 $() 获取命令输出结果 `` 获取命令输出结果 $(()) 算数拓展运算符 -eq、-ne、-lt、le、-gt、-ge 表示相等、不等于、小于、小于或等于、大于、大于>或等于 [] 条件表达式 -d 是否是目录，如 if [-f a.sh] -f 是否是普通文件，如if [-d a.txt] -a、-o 逻辑和与逻辑或
